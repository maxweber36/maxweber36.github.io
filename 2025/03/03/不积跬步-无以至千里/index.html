独立开发
不积跬步,无以至千里
Mar 3, 2025

今天开始学习 idoubi 的 AI Wall Paper 的项目。得益于之前的 react 和 nextjs 的学习，这次学习起来，没有第一次那么懵逼了。但是对于自己来说，还是有很多需要自己需要补充的知识。今天开始学习 tailwindcss。这个是一个非常有用的。

今天有哪些心得：

构建产品，独立开发，是一个持久而需要长期投入的一个事情。
写代码，不仅仅是要写能够运行的代码，重要的还有要写能够便于扩展的代码，能够便于维护的代码。
构建产品是一个边际成本递减的事情，写代码也是同样如此。因此写代码不能仅仅考虑眼前的事情，还要多多想想未来。
做独立开发，不仅仅只是在写代码层面上提炼自己，还需要在分析需求上，产品运营上，价值交付上，商业模式上，都需要多多思考和学习。

今天完成了什么？

  1. 开始 idoubi 的 AI WallPaper 的项目。
  2. 今天学习了如何 基于 NextJS 构建 API。
  3. 学习了如何基于 Rest Client 插件进行 Debug。但是对于 Prettier Code 的插件，如何使用，还是不太熟悉。
